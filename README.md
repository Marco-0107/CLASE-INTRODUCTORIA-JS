# CLASE INTRODUCTORIA JAVASCRIPT

Material base para una clase introductoria de JavaScript orientada a practica.

## Estructura

- `clase/00-hola-mundo.js`
- `clase/01-tipos-de-datos.js`
- `clase/02-operadores-y-conversion.js`
- `clase/03-condicionales.js`
- `clase/04-bucles.js`
- `clase/05-funciones.js`
- `clase/06-arrays-y-objetos.js`
- `clase/07-metodos-utiles.js`
- `clase/08-scope-y-hoisting.js`
- `clase/09-mini-reto.js`
- `clase/10-ejercicios-propuestos.js`

## Orden sugerido para dictar

1. Variables y salida por consola
2. Tipos de datos
3. Operadores y conversion
4. Condicionales
5. Bucles
6. Funciones
7. Arrays y objetos
8. Metodos utiles
9. Scope y hoisting
10. Mini reto guiado
11. Ejercicios propuestos

## Como ejecutar

Desde la raiz del proyecto, ejecuta cualquier archivo con Node:

```bash
node clase/00-hola-mundo.js
node clase/01-tipos-de-datos.js
```

Tip: puedes abrir dos terminales, una para explicar y otra para ejecutar ejemplos en vivo.

## Objetivo de la clase

Al terminar, el estudiante deberia poder:

- Declarar variables con `let` y `const`
- Identificar tipos de datos basicos
- Usar condicionales y bucles
- Escribir y llamar funciones
- Manipular arrays y objetos
- Resolver problemas simples con JavaScript
