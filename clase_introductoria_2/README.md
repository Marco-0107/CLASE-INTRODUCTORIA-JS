# CLASE INTRODUCTORIA 2 - Arrays y CRUD simulado

Esta clase refuerza el uso de arreglos para guardar informacion y practicar operaciones tipo CRUD (Crear, Leer, Editar, Eliminar) sin base de datos ni servidor.

## Estructura

- `clase_introductoria_2/clase-arrays-crud.js`
- `clase_introductoria_2/tarea/codigo-base.js`
- `clase_introductoria_2/tarea/README.md`

## Objetivo de la clase

Al terminar esta clase, el estudiante deberia poder:

- Crear arreglos de datos simples y de objetos.
- Guardar informacion dentro de un arreglo.
- Leer/listar informacion guardada.
- Editar un elemento del arreglo por `id`.
- Eliminar un elemento del arreglo por `id`.
- Entender la logica de un CRUD simulado en memoria.

## Como ejecutar

Desde la raiz del proyecto:

```bash
node clase_introductoria_2/clase-arrays-crud.js
node clase_introductoria_2/tarea/codigo-base.js
```

## Sugerencia para dictar (45-60 min)

1. Repaso rapido de arrays (5 min)
2. Arreglo de objetos: estructura de datos (10 min)
3. Crear y listar elementos (10 min)
4. Editar y eliminar por `id` (15 min)
5. Explicar tarea y resolver dudas (10-20 min)
